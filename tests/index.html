ShowContextMenu Test

This is a manual test to check the functionality of the ShowContextMenu add-on.

Preconditions

  • NVDA is running on the system.
  • The ShowContextMenu add-on is installed in NVDA.

Steps

  1. Open this HTML file in target browser.
  2. Go to the section titled "Test Rig".
  3. Set system focus to the "Test Link" link using the Tab key.
  4. Set the NVDA cursor to the "Test Image" image using the arrow keys or the G key.
  5. Make sure system focus remains on the link using NVDA+Tab command.
  6. Press the command of ShowContextMenu add-on (NVDA+Shift+F10 or NVDA+Application key).
  7. Examine the context menu items.

Expected Result

  • The context menu for the image opens. It should contain items like "Copy image address" or something similar.
