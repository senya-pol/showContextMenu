# Дополнение ShowContextMenu для NVDA

**ShowContextMenu** позволяет отображать контекстные меню для текущих элементов веб-контента в режиме обзора, даже если их невозможно сфокусировать с помощью клавиатуры.

## Возможности

* Работает с элементами, реализующими **IAccessibleAction**.
* Открывает контекстные меню с помощью сочетаний клавиш:
	+ **NVDA+Shift+F10**
	+ **NVDA+Клавиша контекстного меню**
* Протестировано с NVDA 2019.3 и более поздними (до 2025.3).

## Установка

1. Скачать файл `.nvda-addon` со [страницы релизов на GitHub](https://github.com/senya-pol/showContextMenu/releases).  
2. Открыть меню NVDA → **Сервис → Магазин дополнений → Установить из стороннего источника** → выбрать загруженный файл дополнения.  
3. Перезагрузить NVDA для активации дополнения.

## Использование

* Установите фокус NVDA на любой элемент веб-контента в режиме обзора (пока поддерживаются только браузеры на основе Chromium).
* Нажмите **NVDA+Shift+F10** или **NVDA+Клавиша контекстного меню** для отображения контекстного меню.
* Вы можете переназначить жест через меню NVDA → **Параметры → Жесты ввода → Режим обзора → Показывает контекстное меню для текущего элемента веб-контента**
