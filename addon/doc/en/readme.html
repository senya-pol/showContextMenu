ShowContextMenu NVDA Add-on

ShowContextMenu allows you to open context menus for focused elements in Chromium-based browsers, including Yandex, Chrome, Edge, and others.

Features

  • Works with elements that implement IAccessibleAction.
  • Open context menus using keyboard shortcuts:
    • NVDA+Shift+F10
    • NVDA+Applications key
  • Tested with NVDA 2019.3 and later (up to 2025.3).

Installation

  1. Download the .nvda-addon file from the GitHub Releases page.
  2. Open NVDA → Tools → Add-ons → Install → select the downloaded file.
  3. Restart NVDA to activate the add-on.

Usage

  • Focus any element in a Chromium-based browser.
  • Press NVDA+Shift+F10 or NVDA+Applications key to show the context menu.
